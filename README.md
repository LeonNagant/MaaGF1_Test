## 🚀 快速开始

### 用户使用指南

1. 下载最新[发行版](https://github.com/LeonNagant/MaaGF1_Test/releases)  
   `MaaGF1-GUI-v1.x.xx-x86_64-.zip`
2. 解压文件
3. （管理员）运行 `MFAAvalonia.exe`
4. 选择「少女前线」窗口
5. 选择脚本，查看并调整对应设置
6. 点击开始运行
